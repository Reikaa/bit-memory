Detailed Description

template<typename BlockAllocator>
class bit::memory::cached_block_allocator< BlockAllocator >

This class is an adapter around a given block allocator that caches any allocated blocks for reuse.

Any constructors for the underlying BlockAllocator are propagated to the cache_block_allocator constructor – allowing it to be constructed as though it is the underlying type.
