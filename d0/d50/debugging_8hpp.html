void* bit::memory::debug_untag_bytes ( void *  p,
std::size_t  n,
debug_tag  tag,
std::size_t *  stomped 
)

Untags memory previously tagged with debug_tag bytes.

This function validates that the bytes previously tagged still contain the same tag (thus no buffer-overflow conditions have occurred).

If a buffer overflow has occurred, then this function returns a pointer to the first byte that has been overwritten, and stomped will now contain the number of bytes that have been modified.

Otherwise this function returns nullptr, and stomped remains unchanged

Parameters
ppointer to the memory to untag
nthe number of bytes to untag
tagthe expected debug tag
[out]stompedoutputs the number of bytes stomped if a buffer overflow occurs; otherwise remains unchanged
Returns
pointer to the first byte that does not match tag (if modified); otherwise returns nullptr
