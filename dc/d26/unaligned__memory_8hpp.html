◆ load_unaligned()

template<typename T >
T bit::memory::load_unaligned ( const void *  p)

Loads an arbitrary type T from unaligned memory.

Note
Only types that don't contain pointers that point inside the loaded type can be safely loaded. Otherwise, the pointers will be pointing to potentially unsafe memory. Ideally, this function should only be used to load fundamental types, or simple pod/aggregate types.
Parameters
ppointer to the memory to load from
Returns
the type T
