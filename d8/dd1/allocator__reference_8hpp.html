Detailed Description

This header defines a manner to wrap allocators in a reference-like type.

This allows allocators to be shared and accessed from multiple locations at a single time, without requiring pointer-semantics. allocator_traits and block_allocator_traits both work with allocator references and will behave as though it is the underlying allocator.
