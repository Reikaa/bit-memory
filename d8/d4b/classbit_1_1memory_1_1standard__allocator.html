Detailed Description

template<typename T, typename AllocatorStorage>
class bit::memory::standard_allocator< T, AllocatorStorage >

The adapter to make an Allocator behave like a standard allocator.

This requires an instance of an AllocatorStorage type to hold onto the underlying allocator, and possibly contribute to the lifetime (depending on the storage policy)

Template Parameters
Tthe underlying allocator type
AllocatorStoragethe storage that holds the allocator
