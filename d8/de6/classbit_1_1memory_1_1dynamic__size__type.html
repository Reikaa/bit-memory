Detailed Description

template<std::size_t Index, std::size_t Size>
class bit::memory::dynamic_size_type< Index, Size >

This type is used as a compile-time constant for storing size values.

If Size is dynamic_size, then this type holds a data member.

Template Parameters
Indexthe index of the value (used for unique identification in ebo)
