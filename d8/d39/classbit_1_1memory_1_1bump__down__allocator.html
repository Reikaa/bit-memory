Detailed Description

An allocator that allocates memory in contiguous memory by 'bumping' the head pointer to lower memory address.

Allocations are distributed in a decreasing memory-address pattern.

This allocator offers 2 options for deallocations:

  • Truncated deallocations, managed through deallocate_all
  • LIFO deallocations, through deallocate

The two forms of deallocations are incompatible; either truncated deallocations can be used OR LIFO deallocations can be used.

The two deallocation patterns allow for this to act as a simple linear/contiguous temporary allocator, or it can be used as a stack-based allocator.
