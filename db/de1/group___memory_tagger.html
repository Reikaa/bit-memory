Detailed Description

This concept defines a manner of tagging allocations for better visibility in a debugger.

For type T to be MemoryTagger, it must satisfy the following:

Provided

  • T - a MemoryTagger type
  • t - an instance of type T
  • s - the size of an allocation
  • p - a void pointer

The following expressions must be well-formed and with the following side-effects:

t.tag_allocation( p, s );

T tags the allocation at address p with size s

t.tag_deallocation( p, s );

T tags the deallocation at address p with size s
