Detailed Description

A non-owning allocator type that type-erases an underlying allocator.

This view is non-owning, and thus does not participate in the lifetime of a given allocator.

Note
Since this is type-erased, any static information – such as is_stateless, is_always_equal, etc – is lost. It is NOT intended as a drop-in replacement for allocators; only a means of type-erasing the allocator so that underlying APIs need-not require the type of allocator embedded in the class.
