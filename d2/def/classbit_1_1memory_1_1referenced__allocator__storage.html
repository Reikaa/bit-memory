Detailed Description

template<typename Allocator>
class bit::memory::referenced_allocator_storage< Allocator >

A storage type for holding onto allocators by a reference.

The lifetime of the underlying referenced allocator is the same as the original allocator being referenced; meaning that this storage type should be treated carefully to avoid dangling references.
